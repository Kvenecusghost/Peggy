ISR ( INT6_vect  )

Прерывание вызывается при получении сигнала по LoRa. Таймер 0 останавливается. Если сообщение является конфигурационным, то модуль Specto-A отправляет полученное сообщение обратно в эфир в виде эхо. Далее вызывается функция реконфигурации модуля Specto-A. Таймер 0 снова запускается.
